# 1、上传安装包

使用 SFTP 工具将压缩包上传到 /opt/software

# 2、解压文件

将压缩包解压到 /opt/module/

[laowang@hadoop102 software]$ tar -zxvf apache-flume-1.9.0-bin.tar.gz -C /opt/module/

# 3、修改名称

将 apache-flume-1.9.0-bin 目录名称修改为 flume-1.9.0

[laowang@hadoop102 software]$ cd /opt/module/
[laowang@hadoop102 module]$ mv apache-flume-1.9.0-bin flume-1.9.0

# 4、删除文件

将 /opt/module/flume-1.9.0/lib 目录下的 guava-11.0.2.jar 删除，主要是为了兼容 Hadoop 3.1.3

[laowang@hadoop102 module]$ cd flume-1.9.0/lib/
[laowang@hadoop102 lib]$ rm guava-11.0.2.jar

也可以不删除，将文件名进行修改，如修改成 guava-11.0.2.bak

# 5、配置环境变量

[laowang@hadoop102 lib]$ vim /etc/profile.d/my_env.sh

添加如下内容：

# FLUME_HOME
export FLUME_HOME=/opt/module/flume-1.9.0
export PATH=$PATH:$FLUME_HOME/bin

刷新环境变量

[laowang@hadoop102 lib]$ source /etc/profile
