# 一、单机环境搭建

# 1、准备工作

Cent OS取消打开文件数限制：在 limits.conf 文件的末尾加入以下内容

[laowang@hadoop102 ~]$ sudo vim /etc/security/limits.conf
* soft nofile 65536
* hard nofile 65536
* soft nproc 131072
* hard nproc 131072

Cent OS取消打开文件数限制：在 20-nproc.conf 文件的末尾加入以下内容

[laowang@hadoop102 ~]$ sudo vim /etc/security/limits.d/20-nproc.conf
* soft nofile 65536
* hard nofile 65536
* soft nproc 131072
* hard nproc 131072

安装依赖

[laowang@hadoop102 ~]$ sudo yum install -y libtool
[laowang@hadoop102 ~]$ sudo yum install -y *unixODBC*

Cent OS取消SELINUX

[laowang@hadoop102 ~]$ sudo vim /etc/selinux/config
SELINUX=disabled

配置另外两台服务器：将 hadoop102 配置分发

[laowang@hadoop102 ~]$ sudo xsync /etc/security/limits.conf
[laowang@hadoop102 ~]$ sudo xsync /etc/security/limits.d/20-nproc.conf
[atguigu@hadoop102 ~]$ sudo xsync /etc/selinux/config

配置另外两台服务器：在 hadoop103、 hadoop104 上安装依赖

[laowang@hadoop103 ~]$ sudo yum install -y libtool
[laowang@hadoop103 ~]$ sudo yum install -y *unixODBC*

[laowang@hadoop104 ~]$ sudo yum install -y libtool
[laowang@hadoop104 ~]$ sudo yum install -y *unixODBC*

# 2、正式安装

在 hadoop102 的 /opt/software 目录下创建 clickhouse 目录

[laowang@hadoop102 software]$ mkdir clickhouse

将安装包上传到 clickhouse 目录下，并将目录同步至 hadoop103 、hadoop104

[laowang@hadoop102 software]$ xsync clickhouse

分别在三台机子上安装这4个rpm文件

[laowang@hadoop102 clickhouse]$ sudo rpm -ivh *.rpm
[laowang@hadoop103 clickhouse]$ sudo rpm -ivh *.rpm
[laowang@hadoop104 clickhouse]$ sudo rpm -ivh *.rpm

查看安装情况

[laowang@hadoop102 clickhouse]$ sudo rpm -qa|grep clickhouse

修改配置文件

[laowang@hadoop102 clickhouse]$ sudo vim /etc/clickhouse-server/config.xml

把 <listen_host>::</listen_host> 的注释打开，这样的话才能让ClickHouse被除本机以外的服务器访问

分发配置文件

[laowang@hadoop102 clickhouse]$ sudo xsync /etc/clickhouse-server/config.xml

启动 hadoop102 的 Server

[laowang@hadoop102 clickhouse]$ sudo systemctl start clickhouse-server
